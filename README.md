# Comiketter

コミケ向けX（旧Twitter）専用Chrome拡張機能

## 📦 導入手順

### 1. 拡張機能のダウンロード

1. [Comiketter Releases](https://github.com/Hutaba-Makura/Comiketter/releases) ページにアクセス
2. **Assets** セクションから **Comiketter-v1.0.0.zip** をダウンロード
3. 任意のフォルダに保存

### 2. Chromeへのインストール

詳細な手順は [Chrome拡張機能のインストール方法](https://note.com/cute_echium873/n/n997dcf40b3a1) をご参照ください。

**簡単な手順：**
1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. ダウンロードしたzipファイルを解凍したフォルダを選択

### 3. 使用方法

インストール後、X（旧Twitter）のページで拡張機能が自動的に動作します。

本リポジトリは、一部機能を [TwitterMediaHarvest](https://github.com/EltonChou/TwitterMediaHarvest) を参考に作成しており、該当箇所には出典表記を記載しています。

## ✨ 主な機能

### CB（カスタムブックマーク）
ツイートのアクションバーに追加されたボタンから、新規にブックマークリストを作成してツイートを追加できます。複数のブックマークリストを作成することで、ブックマークを整理できます。

### メディアダウンロード
ツイートのアクションバーに追加されたボタンから、動画や画像をダウンロードできます。

## 🔜 実装予定の機能

- CBのインポート・エクスポート機能
- ダウンロードしたメディアの履歴閲覧機能
- 自動ダウンロード機能・自動CB登録機能
- メディアの一括ダウンロード機能
